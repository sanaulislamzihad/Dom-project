What is the DOM in JavaScript?

TThe Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node corresponds to an HTML element. JavaScript can interact with the DOM to manipulate content, structure, and styles dynamically.

How can I manipulate the DOM using JavaScript?

You can manipulate the DOM using JavaScript by selecting elements with methods like getElementById(), querySelector(), or getElementsByClassName(). Once selected, you can modify properties like innerHTML, style, and attributes or even add/remove elements using methods such as appendChild(), removeChild(), or createElement()

What is the difference between innerHTML and textContent?

innerHTML returns or sets the HTML content inside an element, including any HTML tags, whereas textContent only returns or sets the plain text within an element. Use textContent when you want to prevent HTML injection or avoid rendering HTML inside an element

How do you manipulate an element's attributes and styles useing the DOM?

To manipulate an element's attributes and styles using the DOM I can use : Attributes: getAttribute(), setAttribute(), removeAttribute() Styles: style property, className property, classList property, setProperty(), getPropertyValue(), removeProperty()
